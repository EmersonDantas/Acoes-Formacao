# Material Design

Esta vídeo aula é uma introdução sobre Material Design. Fala sobre quem criou e o que ele é, seus objetos como metodologia de design, seus princípios, as características, exemplos de aplicativos que utilizam essa metodologia e um breve tutorial de como desenvolver um simples aplicativo utilizando esse conceito.
